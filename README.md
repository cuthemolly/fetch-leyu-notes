# fetch-leyu-notes

本仓库用于归档和发布多个独立的 HTML 页面，作为网页资源的集中存储与展示空间。

## 项目简介

该项目不针对任何特定域名或网站，仅作为通用 HTML 页面存档库。所包含的页面内容可能涉及各类信息整理、笔记、工具页面等，所有页面均为独立静态文件，可直接通过浏览器访问。

## 目录说明

```
.
├── index.html          # 仓库主页索引（可选）
├── pages/              # 归档页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面文件。
- `assets/`：存放页面可能使用的公共样式、脚本或媒体资源。

## 页面归档说明

- 每个 HTML 页面应为独立、完整的静态文件，可直接在浏览器中打开。
- 页面内容不限于特定领域或用途，但需遵守开源社区规范。
- 建议在页面内使用相对路径引用资源，以保证在仓库内正常显示。

## 维护说明

- 欢迎通过 Pull Request 提交新的归档页面，或对现有页面进行修正。
- 提交前请确保页面无外部依赖（或已在 `assets/` 中提供），且不包含恶意代码。
- 如发现页面功能异常或内容错误，欢迎提交 Issue 反馈。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/fetch-leyu-notes.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看内容。
3. 也可通过 GitHub Pages 等静态托管服务部署本仓库，获得在线访问地址。
